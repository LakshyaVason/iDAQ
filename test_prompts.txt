### Test 2: Component Understanding
**Prompt:**
```
Why would you use an IGBT instead of a MOSFET in a high-power inverter application?
```

**Expected Response:**
- IGBT advantages: lower conduction losses at high power
- MOSFET advantages: faster switching, lower gate drive requirements
- Trade-offs based on voltage/current ratings

---

### Test 3: Fault Mechanism
**Prompt:**
```
If I'm seeing gradual voltage droop on the DC bus with rising MOSFET junction temperature, what components should I check first?
```

**Expected Response:**
- DC link capacitor ESR degradation
- Gate driver supply voltage
- Thermal interface/cooling path
- Should prioritize capacitors first

---

### Test 4: Protection Schemes
**Prompt:**
```
What's the difference between cycle-by-cycle current limiting and overcurrent shutdown in a power converter?
```

**Expected Response:**
- Cycle-by-cycle: limits each switching cycle, automatic recovery
- Overcurrent shutdown: latched fault, requires reset
- Use cases for each method
